The file must use these exact names for the columns: firstName, lastName, email, companyName, country*, status, tags, companyDomain, companyLogo, profileQuality, created, answeredByEmail

You can also add questions identifiers as columns names

Bold columns are mandatory, other columns are optional. The order of the columns does not matter

*should be a valid country code. If not possible it could country name in French, Italian, English, Spanish or Deutsch.
