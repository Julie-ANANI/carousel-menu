Form

Form provides the most common control styles used in forms, including input, textarea, select, checkbox, radio and switch.

Input


        <div class="form-group">
          <label class="form-label" for="name">Name</label>
          <input type="text" class="form-input" placeholder="Name" id="name">
        </div>

        <div class="form-group">
          <label class="form-label" for="last-name">Last Name</label>
          <input type="text" class="form-input" placeholder="Last Name" id="last-name">
        </div>

Textarea

If you want to resize the textarea use the is-resize class to the <textarea> element.


        <div class="form-group">
          <label class="form-label" for="textarea">Textarea</label>
          <textarea rows="3" class="form-input" placeholder="textarea example" id="textarea"></textarea>
        </div>

        <div class="form-group">
          <label class="form-label" for="textarea-2">Textarea 2</label>
          <textarea class="form-input is-resize" id="textarea-2" placeholder="textarea example" rows="3"></textarea>
        </div>

Select


        <div class="form-group">
          <label class="form-label" for="select">Select</label>
          <select class="form-select" id="select">
            <option>Choose an option</option>
            <option>Slack</option>
            <option>Skype</option>
            <option>Hipchat</option>
          </select>
        </div>

Radio

By default they are displayed in inline but if you want to display one per row then use is-full with the form-radio class.


        <div class="form-group">
          <label class="form-label">Default</label>
          <label class="form-radio">
            <input type="radio" name="gender" checked>
            <i class="form-icon"></i> Male
          </label>
          <label class="form-radio">
            <input type="radio" name="gender">
            <i class="form-icon"></i> Female
          </label>
        </div>

        <div class="form-group">
          <label class="form-label">Full</label>
          <label class="form-radio is-full">
            <input type="radio" name="gender" checked>
            <i class="form-icon"></i> Male
          </label>
          <label class="form-radio is-full">
            <input type="radio" name="gender">
            <i class="form-icon"></i> Female
          </label>
        </div>
