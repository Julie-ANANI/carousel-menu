Icons

Place the icon in the icon-container class. Because the icons can take a few seconds to load, and because you want control over the space the icons will take, you can use the icon-container class as a reliable square container that will prevent the page to "jump" on page load.

Some of the basic icons are mentioned below. You can also place the Font Awesome icon inside the icon-container class. To have the background overlay effect use the is-overlay class with the icon-container class.

To use the predefined below icons you have to place those classes with the icon class on the <i> element.

Sizes

There are 6 sizes available. By default it is is-normal. Use is-xs, is-sm, is-normal,is-md, is-lg or is-xl class with the icon-container class.


        <div class="icon-container is-xs is-overlay">
          <i class="icon icon-plus"></i>
        </div>

        <div class="icon-container is-sm is-overlay">
          <i class="icon icon-plus"></i>
        </div>

        <div class="icon-container is-overlay">
          <i class="icon icon-delete"></i>
        </div>

        <div class="icon-container is-md is-overlay">
         <i class="icon icon-menu"></i>
        </div>

        <div class="icon-container is-lg is-overlay">
          <i class="fas fa-user-check"></i>
        </div>

        <div class="icon-container is-xl is-overlay">
          <i class="fas fa-user-times"></i>
        </div>

Tooltip

To show the tooltip place the <span> element inside the icon-container class and use the tooltip class on the <span> element.


        <div class="icon-container is-overlay">
          <span class="tooltip" [attr.data-tooltip]="Delete tooltip">
            <i class="icon icon-delete"></i>
          </span>
        </div>

Disabled

To disabled the icon use the disabled class with the icon-container class. It will hide the tooltip if there is.


        <div class="icon-container is-overlay disabled">
          <span class="tooltip" [attr.data-tooltip]="Delete tooltip">
            <i class="icon icon-delete"></i>
          </span>
        </div>
