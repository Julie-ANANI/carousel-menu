Icons

The icon element is a container for any type of icon font. Because the icons can take a few seconds to load, and because you want control over the space the icons will take, you can use the icon class as a reliable square container that will prevent the page to "jump" on page load.

Default colors


        <div class="icon"><i class="fas fa-user"></i></div>
        <div class="icon is-primary"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-secondary"><i class="fas fa-users"></i></div>
        <div class="icon is-info"><i class="fas fa-user-plus"></i></div>
        <div class="icon is-draft"><i class="fas fa-user-edit"></i></div>
        <div class="icon is-success"><i class="fas fa-user-cog"></i></div>
        <div class="icon is-delete"><i class="fas fa-user-times"></i></div>
        <div class="icon is-white"><i class="fas fa-user-cog"></i></div>

Sizes

The icon comes in 5 sizes: small is-sm, default is-normal, medium is-md, large is-lg and extra large is-xl.


        <div class="icon is-primary is-sm"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-primary is-normal"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-primary is-md"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-primary is-lg"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-primary is-xl"><i class="fas fa-user-friends"></i></div>

Overlay icon

To show the background overlay effect when hover over the icon use is-overlay class with the icon class. To make the background always visible use the is-overlay-active with the icon is-overlay class.


        <div class="icon is-draft is-overlay is-sm"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-draft is-overlay"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-draft is-overlay is-md"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-draft is-overlay is-lg"><i class="fas fa-user-friends"></i></div>
        <div class="icon is-draft is-overlay is-xl"><i class="fas fa-user-friends"></i></div>
