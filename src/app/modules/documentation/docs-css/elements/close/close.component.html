Close

A versatile close cross. The close element is a stand-alone element that can be used in different contexts. On its own, it's a simple circle with a cross. By default the cross is in white when you hover over it, it shows the overlay background.


        <div class="close"></div>

To show the cross icon with the background overlay use the is-overlay class with the close class.


        <div class="close is-overlay"></div>

Sizes

It comes in 4 different sizes.


        <div class="close is-overlay is-xs"></div>
        <div class="close is-overlay is-sm"></div>
        <div class="close is-overlay is-normal"></div>
        <div class="close is-overlay is-lg"></div>
