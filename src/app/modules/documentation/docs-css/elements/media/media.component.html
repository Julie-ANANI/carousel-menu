Media

Media includes responsive images, figures and video classes.

Images

Add the img-responsive class to <img> element. The image will scale with the parent sizes.


        <figure>
          <img src="https://images.unsplash.com/photo-1497091071254-cc9b2ba7c48a?w=1024" alt="..." class="img-responsive">
        </figure>

Add the img-fit-contain or img-fit-cover class to <img> or <video> elements. The media will crop itself to fit inside the element (and you don't need another container). This feature can replace the classic background image trick.


          <figure>
            <img src="https://images.unsplash.com/photo-1519389950473-47ba0277781c ..." alt="..." class="img-fit-contain">
          </figure>
          <figure>
            <img src="https://images.unsplash.com/photo-1499951360447-b19be8fe80f5 ..." alt="..." class="img-fit-cover">
          <figure>

To specify the image of fixed height and width include the width and height attribute in the <img> element.


            <figure>
              <img src="https://images.unsplash.com/photo-1519389950473-47ba0277781c ..." alt="..." width="320" height="200">
            </figure>

To show the overlay effect over the image when hover over it use the img-overlay class to the <figure> element. Sometimes we do encounter problems while fetching the images and at the same time we want the image container of fixed so in that case make the <figure> element of fixed height.


            <figure class="img-overlay">
              <img src="https://images.unsplash.com/photo-1460925895917-afdab827c52f" class="img-fit-cover" alt="..." width="320" height="200">
            </figure>

To display the button on the center of the image at the time of hover use the img-hover-button class to the <figure> element and to the <span> element i.e. that will be the button use the img-button class to it. But at the same time you have to make figure or either the wrapper of fixed width and height.


            <figure class="img-hover-button">
              <img src="https://images.unsplash.com/photo-1485988412941-77a35537dae4" class="img-fit-cover" alt="..." width="320" height="200">
              <span class="img-button">Learn more</span>
            </figure>

Figure with caption

You can use the element <figure> for an image with a caption. Add the figure class to <figure> element. The images with the img-responsive class will be responsive. And the included class figure-caption will provide basic style for caption. Also, you can use text-left, text-center and text-right for caption alignment.


        <figure class="figure">
          <img src="https://images.unsplash.com/photo-1497091071254-cc9b2ba7c48a?w=1024" class="img-responsive" alt="...">
          <figcaption class="figure-caption text-center">person using track pad</figcaption>
        </figure>

Videos

For responsive video, add a container with the video-responsive class. Insert any YouTube, Vimeo or other iframe/embed video inside the container. The ratio is 16:9 by default. You may add is-responsive-4-3 for 4:3 ratio video container or is-responsive-1-1 for 1:1 ratio.

For responsive <video> elements, you can add the is-responsive class directly.


          <div class="video-responsive is-responsive-4-3">
            <iframe src="..." frameborder="0" allowfullscreen width="..." height="..." >
          </div>

          <video class="video-responsive" src="..."> ... </video>
