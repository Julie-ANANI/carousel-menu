Labels

Labels are formatted text tags for highlighted, informative information. Add the label class to <span> or <small> elements.

Default labels

Default label Black label Primary label Secondary label Progress label Tag label Answer label Info label Draft label Success label Danger label 

        <span class="label">Default label</span>
        <span class="label is-black">Black label</span>
        <span class="label is-primary">Primary label</span>
        <span class="label is-secondary">Secondary label</span>
        <span class="label is-progress">Progress label</span>
        <span class="label is-tag">Tag label</span>
        <span class="label is-info">Info label</span>
        <span class="label is-answer">Answer label</span>
        <span class="label is-draft">Draft label</span>
        <span class="label is-success">Success label</span>
        <span class="label is-danger">Danger label</span>

Sizes

The label comes in 3 different sizes: small is-sm, normal i.e. default is-normal, and large is-lg.

Small Normal Large 

        <span class="label is-info is-sm">Small</span>
        <span class="label is-success is-normal">Default</span>
        <span class="label is-danger is-lg">Large</span>

Rounded labels

To rounded the label add the is-rounded class with the label class.

Rounded Rounded 

        <span class="label is-rounded">Rounded</span>
        <span class="label is-draft is-rounded">Rounded</span>

Icon labels

When you want to show the icon inside the label. Use icon-left for the icon at the left side and use icon-right for the icon at the right side.

Icon small Icon default Icon large 

        <span class="label is-sm">
          <i class="fas fa-tag icon-left"></i> Icon small
        </span>
        <span class="label is-draft">
          Icon default <i class="fas fa-times icon-right"></i>
        </span>
        <span class="label is-large is-danger">
          <i class="fas fa-tag icon-left"></i> Icon default <i class="fas fa-times icon-right"></i>
        </span>
