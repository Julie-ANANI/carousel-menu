Button

Button include simple button styles for actions in different types and sizes. The button is an essential element of any design. It's meant to look and behave as an interactive element of your page.

The button class can be used on <a>, <button> or <input> elements for a default button.

There are classes button-primary, button-link and ghost-primary for predefined primary and link buttons.

Default buttons


        <button class="button button-primary">Primary</button>
        <button class="button button-secondary">Secondary</button>
        <button class="button button-edit">Edit</button>
        <button class="button button-draft">Draft</button>
        <button class="button button-success">Success</button>
        <button class="button button-error">Error</button>

Ghost buttons


        <button class="button ghost-primary">Ghost primary</button>
        <button class="button ghost-secondary">Ghost secondary</button>
        <button class="button ghost-edit">Ghost edit</button>
        <button class="button ghost-draft">Ghost draft</button>
        <button class="button ghost-success">Ghost success</button>
        <button class="button ghost-error">Ghost error</button>

Link button

To make the button act as a link add the button-link class to the button class. This class only works with the default button classes not with the ghost button classes.


        <button class="button button-link">Link button</button>

Sizes

The button comes in 3 different sizes: small button-sm, normal i.e. default button-normal, and large button-lg.


        <button class="button button-sm">Small</button>
        <button class="button button-normal">Default</button>
        <button class="button button-lg">Large</button>

Action button

Add the button-action class to the button class to make the button as action by default it is square shape, or add another circle class for a round button.


        <button class="button button-success button-action">
          <i class="fas fa-bars"></i>
        </button>
        <button class="button button-action button-error circle">
          <i class="fas fa-plus"></i>
        </button>

Rounded buttons

To rounded the button add the rounded class with the button class.

Rounded Rounded 

        <a class="button button-primary rounded">Rounded</a>
        <a class="button ghost-draft rounded">Rounded</a>

Icon buttons

When you want to show the icon inside the button. Use icon-left to place the icon at the left side and use icon-right when you want to place the icon on the right side.


        <button class="button button-secondary">
          <i class="fab fa-facebook-f icon-left"></i> Facebook
        </button>
        <button class="button ghost-secondary">
          Facebook <i class="fab fa-facebook-f icon-right"></i>
        </button>
