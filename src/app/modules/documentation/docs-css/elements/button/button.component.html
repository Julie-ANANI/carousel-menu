Button

Button includes simple button styles for actions in different types and sizes. The button is an essential element of any design. It's meant to look and behave as an interactive element of your page.

The button class can be used on <a>, <button> or <input> elements for a default button.

Default buttons


        <button class="button is-primary">Primary</button>
        <button class="button is-secondary">Secondary</button>
        <button class="button is-info">Edit</button>
        <button class="button is-draft">Draft</button>
        <button class="button is-success">Success</button>
        <button class="button is-danger">Danger</button>

Outlined buttons


        <button class="button is-primary is-outlined ">Outlined</button>
        <button class="button is-secondary is-outlined ">Outlined</button>
        <button class="button is-info is-outlined ">Outlined</button>
        <button class="button is-draft is-outlined">Outlined</button>
        <button class="button is-success is-outlined">Outlined</button>
        <button class="button is-danger is-outlined">Outlined</button>

Link button

To make the button act as a link add the is-link class to the button class.


        <button class="button is-link">Link button</button>
        <button class="button is-secondary is-link">Link button</button>

Sizes

The button comes in 3 different sizes: small is-sm, normal i.e. default is-normal, and large is-lg.


        <button class="button is-sm">Small</button>
        <button class="button is-normal">Default</button>
        <button class="button is-lg">Large</button>

Action button

Add the is-action class to the button class to make the button as action by default it is square shape, or add another is-circled class for a round button.


        <button class="button is-success is-action">
          <i class="fas fa-bars"></i>
        </button>
        <button class="button is-action is-danger is-circled">
          <i class="fas fa-plus"></i>
        </button>

Rounded buttons

To rounded the button add the is-rounded class with the button class.

Rounded Rounded 

        <a class="button is-primary is-rounded">Rounded</a>
        <a class="button is-info is-outlined is-rounded">Rounded</a>

Icon buttons

When you want to show the icon inside the button. Use icon-left for the icon at the left side and use icon-right for the icon at the right side.


        <button class="button is-secondary">
          <i class="fab fa-facebook-f icon-left"></i> Facebook
        </button>
        <button class="button is-success is-outlined">
          Facebook <i class="fab fa-facebook-f icon-right"></i>
        </button>

Shadow buttons

When you want to show the shadow when you hover over the button. Add the class is-shadowed with the button class. Not to use it with is-outlined class.


        <button class="button is-primary is-shadowed">Shadowed</button>
        <button class="button is-secondary is-shadowed">Shadowed</button>

Button states

Add the is-active class to the button class to show the active state of the button.


        <button class="button is-primary is-active ">Active</button>
        <button class="button is-info is-rounded is-active is-outlined">Active</button>
        <button class="button is-link is-active">Active</button>

Add the disabled class or disabled attribute for disabled button state style.


        <button class="button is-primary disabled">Disabled</button>
        <button class="button is-secondary is-outlined disabled">Disabled</button>
        <button class="button is-link disabled">Disabled</button>

Loading buttons

A button with loading class can show loading indicator. Since the loading spinner is implemented using the :after pseudo-element, it is not supported by the <input type="submit"> element. Consider using <button type="submit"> instead.


        <button class="button is-primary loading"></button>
        <button class="button is-info is-outlined loading"></button>
        <button class="button is-link loading"></button>

List of buttons

You can now create a list of buttons with the buttons class. If the list is very long, it will automatically wrap on multiple lines, while keeping all buttons evenly spaced.

Save changes Save and continue Cancel 

        <div class="buttons">
          <span class="button is-primary">Save changes</span>
          <span class="button is-secondary">Save and continue</span>
          <span class="button is-draft">Cancel</span>
        </div>
