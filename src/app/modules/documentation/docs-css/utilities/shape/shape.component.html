Shape

Shape utilities are used for changing element shapes. There are two shapes currently available that are circle is-circle class and rounded is-rounded class. Use these classes with the shape class. You can also place the text inside these elements.

circle
rectangle

        <div class="shape is-circle"></div>
        <div class="shape is-rounded"></div>
        <div class="shape is-circle">circle</div>
        <div class="shape is-rounded">rectangle</div>

Sizes

It comes in 5 different sizes.


        <div class="shape is-circle is-xs bg-primary"></div>
        <div class="shape is-circle is-sm bg-secondary"></div>
        <div class="shape is-circle is-md bg-progress"></div>
        <div class="shape is-circle is-normal bg-draft"></div>
        <div class="shape is-circle is-lg bg-success"></div>
