Position

Position utilities are used for useful layout and position things, including clearfix, float and positions.

Clear fix

If an element is taller than the element containing it, and it is floated, it will overflow outside of its container you can use the clearfix class to fix this problem. More info.


        <div class="clearfix"></div>

Position

The position property specifies the type of positioning method used for an element (static, relative, absolute, fixed, or sticky).

Absolute text 

        <div class="relative">
          <span class="absolute" style="top: 0; left: 150px">Absolute text</span>
        </div>
ClassTypeDefault valueDescription
staticpositionstaticElements render in order, as they appear in the document flow.
absolutepositionabsoluteThe element is positioned relative to its first positioned (not static) ancestor element.
fixedpositionfixedThe element is positioned relative to the browser window.
relativepositionrelativeThe element is positioned relative to its normal position, so "left:20px" adds 20 pixels to the element's LEFT position.
stickypositionstickyThe element is positioned based on the user's scroll position.

Float

If you want to float the element left and right in the container use the float-left class to float the element at left and float-right class to float the element at the right of the container.

Left element Right element 

        <div class="clearfix">
          <span class="float-left">Left element</span>
          <span class="float-right">Right element</span>
        </div>

Justify content

The justify-content property is used to align the flex items.

ClassTypeDefault valueDescription
justify-centerpositioncenterItems are positioned at the center of the container.
justify-spacepositionspace-betweenItems are positioned with space between the lines.
justify-startpositionflex-startItems are positioned at the beginning of the container.
justify-endpositionflex-endItems are positioned at the end of the container.
justify-aroundpositionspace-aroundItems are positioned with space before, between, and after the lines.

Align items

The align-items property is used to align the flex items vertically. Vertically aligns the flex items when the items do not use all available space on the cross-axis.

ClassTypeDefault valueDescription
align-centerpositioncenterItems are positioned at the center of the container.

Flex wrap

The flex-wrap property specifies whether the flexible items should wrap or not. If the elements are not flexible items, the flex-wrap property has no effect.

ClassTypeDefault valueDescription
flex-nowrappositionnowrapSpecifies that the flexible items will not wrap.
flex-wrappositionwrapSpecifies that the flexible items will wrap if necessary.

White space

The white-space property specifies how white-space inside an element is handled.

ClassTypeDefault valueDescription
whitespace-nowrappositionnowrapSequences of whitespace will collapse into a single whitespace. Text will never wrap to the next line.
