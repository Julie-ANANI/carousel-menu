Loading

Loading indicator

Loading indicator is used for loading or updating. You can add the loading class to a container for loading status.

Add the is-white class to make the loading indicator of white color. Place this class with loading class.

Add the is-lg class for large size.


        <div class="loading"></div>
        <div class="loading is-lg"></div>
        <div class="loading is-white"></div>

Loading content

Loading content is used for loading data animation effect. Put the loading-content class with the Html elements.


        <h2 class="loading-content"></h2>

        <p class="loading-content"></p>

        <div class="card">
          <div class="card-image loading-content">
            <img src="...">
          </div>
          <div class="card-header">
            <div class="card-title loading-content"></div>
          </div>
          <div class="card-body loading-content"></div>
        </div>
