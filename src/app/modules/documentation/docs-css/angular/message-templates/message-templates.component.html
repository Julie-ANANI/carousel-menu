Message templates

This includes the different message components that we have in the application. Here you will see how these components will be rendered.

Message Template 1

Add content or even html element.

Module name: MessageTemplate1Module.

You can change the backgroundColor, widthMax, and srcImage of this template. They all are the Input properties.


        <app-message-template-1>
          Add content or even html element.
        </app-message-template-1>

Message Template 2

Add content or even html element.

Module name: MessageTemplate2Module.

You can change the srcImage value and make animation value to true to have the animation on the bot. They all are the Input properties.


        <app-message-template-2  [animation]="true">
          Add content or even html element.
        </app-message-template-2>
