Avatars

Avatars are user profile pictures.

You can add the avatar class to <figure> or <img> element. There are 4 additional sizes available, including is-xl(64px), is-lg(48px), is-sm(24px), and is-xs(16px). By default, the avatar size is 32px.

For users who don't have profile pictures, you may use their initials for avatars. Add the avatar class and avatar size class to <figure> or <div> element. The data-initial attribute is the name appear inside the avatar. You can also show initials when avatar image is unavailable or not fully loaded.


        <figure class="avatar is-xl">
          <img src="https://images.unsplash.com/photo-1542065435-d6bc2eac3377">
        </figure>

        <figure class="avatar is-lg" data-initial="UM"></figure>

        <figure class="avatar is-sm" data-initial="UM">
          <img src="https://images.unsplash.com/photo-1542065435-d6bc2eac3377">
        </figure>

Presence

Avatars support presence indicators. You can add an <i> element with the avatar-presence class, and add is-online, is-busy or is-away class for different status colors. The default is gray which means offline.


        <figure class="avatar">
          <img src="https://images.unsplash.com/photo-1542065435-d6bc2eac3377">
          <i class="avatar-presence is-online"></i>
        </figure>
