Cards

Cards are flexible content containers.

Add a container element with the card class. And add child elements with the card-image, card-header, card-body and/or card-footer classes. The card-image can be placed in any position.

To have the shadow effect on the card use the is-shadow class with the card class.

Responsive

Add the is-responsive class with the card class when you want to have multiple cards in one row they will expand and collapse according to the available container and all will have the same height. Also wrap them all in a container and use d-flex and flex-wrap classes so that when not much space is available they will move to the next row.
