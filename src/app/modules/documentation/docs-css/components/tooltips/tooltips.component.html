Tooltips

Tooltips provide context information labels that appear on hover and focus.

Add the tooltip class and the data-tooltip attribute, which contains the tooltip content, to non self closing elements. If you want to hide the tooltip for mobile devices use the is-hide-sm class with the tooltip class.

And add the is-right, is-bottom or is-left class to define the position of a tooltip. By default, the tooltip appears above the element.


      <button class="button is-primary tooltip" data-tooltip="top tooltip example">top tooltip</button>
      <button class="button is-primary tooltip is-right" data-tooltip="right tooltip example">right tooltip</button>

Multiline


        <button class="button is-primary tooltip is-multiline" data-tooltip="This is a tooltip multiline example.">multiline tooltip</button>
