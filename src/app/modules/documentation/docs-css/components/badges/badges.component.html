Badges

Badges are often used as unread number indicators or can be used for notification purpose.

If there is no data-badge or the attribute is not specified, the badge will appear as a dot.

Add the badge class to non self closing elements. And add the data-badge attribute to define the content of a badge. The badge will appear in the top-right direction of the element.

Notifications Notifications Notifications Notifications 

        <span class="badge">Notifications</span>
        <span class="badge" data-badge="2">Notifications</span>

Buttons


        <button class="button is-info badge" data-badge="22">Button</button>

Icon button

When you want to show the icon in the form of badge on the button use the is-icon class with the badge class on the button. By default the icon is invisible you need to add the is-active class to make it visible.


        <button class="button is-primary is-outlined badge is-icon">
          <i class="fas fa-save"></i>
          Save
        </button>

        <button class="button is-primary is-outlined badge is-icon is-active">
          <i class="fas fa-save"></i>
          Save
        </button>
