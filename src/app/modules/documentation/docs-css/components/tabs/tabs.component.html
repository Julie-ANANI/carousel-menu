Tabs

Simple responsive horizontal navigation tabs, with different styles.

Add a container element with the tab class. And add child elements with the tab-item class. Inside the tab-item element add the <a> element. The default tabs style has a single border at the bottom. Add the active class with the tab-item class or <a> element to make it highlighted.

  • Tab 1
  • Tab 2
  • Tab 3
  • Tab 4

      <ul class="tab">
        <li class="tab-item active">
          <a>Tab 1</li>
        </li>
        <li class="tab-item">
          <a>Tab 2</li>
        </li>
        ...
      </ul>

Fullwidth

If you want the tabs to take up the whole width available, use the is-fullwidth class.

  • Tab 1
  • Tab 2 

      <ul class="tab is-fullwidth">
        <li class="tab-item active">
          <a><i class="fas fa-pencil-alt"></i>Tab 1</li>
        </li>
        <li class="tab-item">
          <a><i class="icon icon-plus"></i>Tab 2</li>
        </li>
      </ul>
