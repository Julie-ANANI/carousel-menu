Banners

Banners are often used for text notification purpose. You can show it anywhere either on the top of the page or the section. To use it include banner class to <div> element.

This is the example of the default banner.
This is the example of the success banner.
This is the example of the alert banner.

        <div class="banner">This is the example of the default banner.</div>
        <div class="banner bg-success">This is the example of the success banner.</div>
        <div class="banner bg-alert">This is the example of the alert banner.</div>
