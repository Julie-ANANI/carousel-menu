Pagination

A responsive, usable, and flexible pagination.

Add a container element with the pagination class. Add the element with the pagination-previous-control class for the previous button and the pagination-next-control class for the next button. Add a container element in between these two button with the pagination-list class and add the child element with the pagination-link class.

The pagination-link with the active class will be highlighted. You can add the disabled class to the pagination-link or pagination-previous-control or pagination-next-control to have disabled effect.
