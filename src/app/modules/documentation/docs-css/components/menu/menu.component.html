Menu

A simple menu, for any type of vertical navigation.

Add a container element with the menu class. And add child elements with the menu-item class. You can separate menu items with a divider.

There are available in 3 sizes: is-xs i.e. 180px, is-md i.e. 230px and is-lg i.e. 300px. Add one of these classes with the menu class to restrict the menu width. By default it takes the size of the parent container.
