Bars

Bars represent the progress of a task or the value within the known range.

Add a container element with the bar class. And add child elements with the bar-item class. The width percentage value is needed for every bar-item.

There is the is-xs and is-sm class for thinner Bars. Also, you could use Tooltips for any bar-item.


      <div class="bar is-xs">
        <div class="bar-item tooltip" data-tooltip="50" style="width: 50%"></div>
      </div>

      <div class="bar is-sm">
        <div class="bar-item tooltip bg-draft" data-tooltip="70" style="width: 70%"></div>
      </div>

      <div class="bar">
        <div class="bar-item tooltip bg-success" data-tooltip="90" style="width: 90%"></div>
      </div>

Slider

You can add the bar-slider class to the Bars container. And add child elements with the bar-item class and bar-slider-button inside bar-item. You need to set the bar-item width manually to have the slider point.

If there are two bar-item divs in one bar-slider, you will have a range slider.


      <div class="bar bar-slider">
        <div class="bar-item" style="width: 50%">
          <button class="bar-slider-button button"></button>
        </div>
      </div>

      <div class="bar bar-slider">
        <div class="bar-item" style="width: 25%">
          <button class="bar-slider-button button"></button>
        </div>
        <div class="bar-item" style="width: 75%">
          <button class="bar-slider-button button"></button>
        </div>
      </div>

Scale

Add a container element with the bar-scale class. And add child elements with the bar-item class and is-number-(1-9) class to specify the colors. Add the child elements to the bar-item with the bar-fill, bar-stat, bar-score, and bar-picture class. You need to set the value of the bar-fill, bar-stat, and bar-score manually.
