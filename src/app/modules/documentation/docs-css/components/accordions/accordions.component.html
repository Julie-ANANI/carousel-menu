Accordions

Accordions are used to toggle sections of content.


        <!-- standard Accordion example -->
        <div class="accordion">
          <input type="checkbox" id="accordion-1" name="accordion-checkbox" hidden>
          <label class="accordion-header" for="accordion-1">
            <i class="fas accordion-icon fa-chevron-right"></i>
            Title
          </label>
          <div class="accordion-body">
          <!-- Accordion content -->
          </div>
        </div>

        <!-- mutually exclusive Accordion example (with same input names) -->
        <div class="accordion">
          <input type="radio" id="accordion-1" name="accordion-radio" hidden>
          <label class="accordion-header" for="accordion-1">
            Title
          </label>
          <div class="accordion-body">
          <!-- Accordion content -->
          </div>
        </div>

Alternatively, you can use details and summary instead of input radio or checkbox trick. Add the open attribute to details to expand it.


      <!-- details and summary Accordion example -->
      <details class="accordion" open>
        <summary class="accordion-header">
          <i class="fas accordion-icon fa-chevron-right"></i>
          Title
        </summary>
        <div class="accordion-body">
        <!-- Accordion content -->
        </div>
      </details>
