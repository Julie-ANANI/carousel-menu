Dropdown

The dropdown is a combination of buttons and menus. Dropdown menus component is built entirely in CSS. It is triggered by :focus event.

Add a container element with the dropdown class. And add a focusable element with the dropdown-toggle class for the button and a menu component right next to it. You also need to add tabindex to make the buttons focusable.

If the dropdown is close to the right edge of the browser, you can add the is-right class to the container to prevent it appearing off screen.

Also, you can implement the JS to interact with the dropdown menus by adding the active class to the dropdown container.

To make the dropdown disable add the disabled class with the dropdown-toggle class.
