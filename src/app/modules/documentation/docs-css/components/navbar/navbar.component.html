Navbar

A responsive horizontal navbar that can support images, links, buttons, and dropdowns.

Add a main container element with the navbar class. Add the navbar-brand class which is at the left side, always visible, which usually contains the logo and optionally some links or icons.

Add the navbar-menu class i.e. the right side. It contains the different navigation options. Add the navbar-item class to the each single item of the navbar, which can either be an a or a div element.

For the dropdown menu add the has-dropdown class with the navbar-item class . Add the dropdown-toggle class to the sibling of a dropdown. Inside a dropdown add the navbar-dropdown class to show the dropdown menu, which can include navbar-item and dividers. Add the navbar-divider class, a horizontal line to separate navbar items.

To make the navbar-item highlighted add the active class to it.

For the avatar dropdown menu you also have to add the has-avatar class. On the small devices the dropdown menu inside in it is hidden.

Add the navbar-burger class i.e. the hamburger icon, which toggles the navbar menu on touch devices. To active the menu using the hamburger icon you have to add the active class with the navbar-menu class.
