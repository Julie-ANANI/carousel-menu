Sidebar

A classic sidebar, in which you can include any content you want.

Add a separate container with the sidebar-backdrop class to show the overlay. Declare it before the main container of the sidebar. Add the active or inactive class to appear and disappear it.

To show the full page sidebar add the main container with the sidebar class. By default the position is left. To have it in right use the is-right class with it. By default is it visible but you can add the active or inactive class to appear and disappear it.

Add the child elements with the sidebar-container. Inside that add the child elements with the sidebar-header and sidebar-body.

To have the different width just play with the width css property of the main container that includes the sidebar class.

Inline

When you want to have the sidebar inside the page. Add the main container with the sidebar-inline. You can also add the above mentioned classes in it.

By default its height is 100vh. To have the different height and width just play with the height and width css property.

Inline sidebar 

        <aside class="sidebar-inline" style="height: 200px">
          Inline sidebar
        </aside>
