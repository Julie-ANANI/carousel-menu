Popovers

They are small overlay content containers.

Wrap an element by a container with the popover class. And add a container with the popover-container next to the element. Use card component class inside the popover-container.

Also, you can add the is-right, is-bottom or is-left class to define the position. By default, the popover appear above the element.

      <div class="popover">
        <button class="button is-primary">top popover</button>
        <div class="popover-container">
          <div class="card">
            This is the example of the top popover.
          </div>
        </div>
      </div>

      <div class="popover is-bottom">
        <button class="button is-primary">bottom popover</button>
        <div class="popover-container">
          <div class="card">
            This is the example of the bottom popover.
          </div>
        </div>
      </div>
