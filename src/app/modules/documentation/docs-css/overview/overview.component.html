Overview

This documentation contains everything you need to know to create a page. It contains both the css classes and also the angular components that we have for the application.

Umi.scss is a lightweight, responsive and modern SCSS framework for faster and extensible development.

It provides basic styles for typography and elements, flexbox based responsive layout system, pure SCSS components and utilities with best practice coding and consistent design language.

To create this document help from the Spectre.css is taken.

Default variables

NameTypeDefault valueVariable for cssDescription
Font size size 1rem $font-size The font size of the text is 1rem and 1rem = 20px.
Font weight weight 400 $font-weight-normal The font weight of the text is 400.
Line height height 1.5 $line-height The line height of the text is 1.5 times of the text font size.
Icon size size 0.75rem $font-icon The default font size for the icon is 0.75rem = 15px.
