Overview

This documentation contains everything you need to know to add css into a page. It includes the different classes and examples of them to make your life easy with the css :)

It's the UMI framework which we uses for all the UMI applications. It is a lightweight, responsive and modern SCSS framework for faster and extensible development.

It contains the unified css for all the applications but its not an end, if you want to changes certain css for elements I encourage you do so in the component not globally.

It provides styles for typography and elements, flexbox based responsive layout system, pure SCSS components and utilities with best practice coding and consistent design language.

To create this document, I took the help from Spectre.css.
