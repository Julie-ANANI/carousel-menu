Buttons

Buttons include simple button styles for actions in different types and sizes. The button is an essential element of any design. It's meant to look and behave as an interactive element of your page.

The button class can be used on <a>, <button> or <input> elements for a default button.

Default


        <button class="button is-primary">Primary</button>
        <button class="button is-white">White</button>
        <button class="button is-dark">Dark</button>
        <button class="button is-warning">Warning</button>
        <button class="button is-draft">Draft</button>
        <button class="button is-info">Info</button>
        <button class="button is-success">Success</button>
        <button class="button is-danger">Danger</button>

Outlined


        <button class="button is-primary is-outlined">Primary</button>
        <button class="button is-white is-outlined">White</button>
        <button class="button is-dark is-outlined">Dark</button>
        <button class="button is-warning is-outlined">Warning</button>
        <button class="button is-draft is-outlined">Draft</button>
        <button class="button is-info is-outlined">Info</button>
        <button class="button is-success is-outlined">Success</button>
        <button class="button is-danger is-outlined">Danger</button>

Link

To make the button act as a link add the is-link class to the button class. It will make the border and background transparent and add the underline under the text when hover.


        <button class="button is-primary is-link">Primary</button>
        <button class="button is-white is-link">White</button>
        <button class="button is-dark is-link">Dark</button>
        <button class="button is-warning is-link">Warning</button>
        <button class="button is-draft is-link">Draft</button>
        <button class="button is-info is-link">Info</button>
        <button class="button is-success is-link">Success</button>
        <button class="button is-danger is-link">Danger</button>

Sizes

The button comes in 5 different sizes: xtra-small: is-xs, small: is -sm, normal i.e. by default, large: is-large, and full-width: is-fullwidth.


        <button class="button is-primary is-xs">Xtra small</button>
        <button class="button is-dark is-sm">Small</button>
        <button class="button is-warning">Normal</button>
        <button class="button is-draft is-lg">Large</button>
        <button class="button is-success is-fullwidth">Full width</button>

Circled

Add the is-circled class to the button class to make the button circled style. You can also use the different button sizes.

The size is-lg not available for this type of button. You can use this style for an action button too.


        <button class="button is-primary is-circled is-xs"><i class="fab fa-facebook-f"></i></button>
        <button class="button is-danger is-circled is-sm"><i class="fab fa-facebook-f"></i></button>
        <button class="button is-success is-circled"><i class="fas fa-plus"></i></button>

Rounded

To rounded the button add the is-rounded class with the button class.


        <button class="button is-primary is-rounded">Primary</button>
        <button class="button is-dark is-rounded">Dark</button>
        <button class="button is-warning is-rounded">Warning</button>
        <button class="button is-draft is-outlined is-rounded">Draft</button>
        <button class="button is-info is-outlined is-rounded">Info</button>
        <button class="button is-success is-outlined is-rounded">Success</button>
        <button class="button is-danger is-outlined is-rounded">Danger</button>

Icon

For the icons you can font-awesome, images, svg or even the predefined icons we have added (here).

When you want to show the icon inside the button. Use icon-left for the icon at the left side and use icon-right for the icon at the right side.


        <button class="button is-secondary">
          <img class="icon-left" src="https://res.cloudinary.com/umi/image/upload/app/default-images/pictos/picto-edit-white.svg">Edit project
        </button>
        <button class="button is-success is-outlined">
        Facebook <i class="fab fa-facebook-f icon-right"></i>
        </button>

Loading


        <button class="button is-primary is-loading"><span class="loading is-white"></span></button>
        <button class="button is-dark is-outlined is-loading"><span class="loading"></span></button>
        <button class="button is-danger is-link is-loading"><span class="loading"></span></button>

States

Add the disabled class or disabled attribute for disabled button state style.


        <button class="button is-primary disabled">Disabled</button>
        <button class="button is-success is-outlined disabled">Disabled</button>
        <button class="button is-link disabled">Disabled</button>
