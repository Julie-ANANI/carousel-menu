Form

Form provides the most common control styles used in forms, including input, textarea, select, checkbox, radio and switch.

Input


              <div class="form-group">
                <label class="form-label" for="name">Name</label>
                <input type="text" class="form-input" placeholder="Name" id="name">
              </div>
              <div class="form-group">
                <label class="form-label" for="last-name">Last Name</label>
                <input type="text" class="form-input" placeholder="Last Name" id="last-name">
              </div>

Textarea

If you want to resize the textarea use the is-resize class to the <textarea> element.


              <div class="form-group">
                <label class="form-label" for="textarea">Textarea</label>
                <textarea rows="3" class="form-input" placeholder="textarea example" id="textarea"></textarea>
              </div>

              <div class="form-group">
                <label class="form-label" for="textarea-2">Textarea 2</label>
                <textarea class="form-input is-resize" id="textarea-2" placeholder="textarea example" rows="3"></textarea>
              </div>

Select


              <div class="form-group">
                <label class="form-label" for="select">Select</label>
                <select class="form-select" id="select">
                  <option>Choose an option</option>
                  <option>Slack</option>
                  <option>Skype</option>
                  <option>Hipchat</option>
                </select>
              </div>

Switch


              <div class="form-group">
                <label class="form-switch">
                  <input type="checkbox">
                  <i class="form-icon"></i> Send me emails with news and tips
                </label>
              </div>

              <div class="form-group">
                <label class="form-switch">
                <input type="checkbox" checked>
                  <i class="form-icon"></i> Send me emails with news and tips
                </label>
              </div>

Radio


              <div class="form-group">
                <label class="form-label">Gender</label>
                <label class="form-radio">
                  <input type="radio" name="gender" checked>
                  <i class="form-icon"></i> Male
                </label>
                <label class="form-radio">
                  <input type="radio" name="gender">
                  <i class="form-icon"></i> Female
                </label>
              </div>

Checkbox


              <div class="form-group">
                <label class="form-checkbox">
                  <input type="checkbox">
                  <i class="form-icon"></i> Remember me
                </label>
                <label class="form-checkbox">
                  <input type="checkbox" checked>
                  <i class="form-icon"></i> Remember me
                </label>
              </div>

You can change checkbox to indeterminate state by setting the indeterminate property of input checkboxes to true. You can't make a checkbox indeterminate through HTML. There is no indeterminate attribute. It is a property of checkboxes though, which you can change via JavaScript.

Inline forms

You can add is-inline class with the form-radio or form-checkbox class to make them inline in one row.


              <div class="form-group">
                <label class="form-radio is-inline"> ... </label>
                <label class="form-radio is-inline"> ... </label>
              </div>

              <div class="form-group">
                <label class="form-checkbox is-inline"> ... </label>
                <label class="form-checkbox is-inline"> ... </label>
              </div>

Sizes

For extra smaller or smaller input and select controls use the is-xs or is-sm class with the form-select and form-input classes.


              <input class="form-input is-sm">

              <select class="form-select is-xs"> ... </select>

Icons

Add the wrapper with the has-icon-left/has-icon-right class to <input> element. And you also need to add a modifier on the icon. icon is-left if has-icon-left is used and icon is-right if has-icon-right is used.


        <div class="form-group has-icon-left has-icon-right">
          <input type="email" class="form-input" placeholder="Email">
          <span class="icon is-left"><i class="icon icon-plus"></i>
          <span class="icon is-right"><i class="icon icon-check"></i>
        </div>

You can use the loading class for loading state.


                  <div class="form-group has-icon-right">
                    <input type="text" class="form-input is-sm" placeholder="Name">
                    <span class="icon is-right loading">
                  </div>

Input group


              <div class="input-group">
                <div class="form-group">
                  <input type="text" class="form-input is-sm" placeholder="First Name">
                </div>
                <div class="form-group">
                  <input type="text" class="form-input is-sm" placeholder="Last Name">
                </div>
              </div>

Validation styles

To use form validation styles, you can either add is-success or is-error class to the controls directly or add has-success or has-error class to parent elements. Use the form-input-hint class to provide form validation success and error messages.

Use the has-success or has-error when you work with the Angular Forms so based on the states of the form the styles will be added.

You can use both classes also together.

The name is valid.

The last name is invalid.

This option is invalid.

This option is valid.


              <div class="form-group">
                <label class="form-label">Input</label>
                <input type="text" class="form-input is-success" placeholder="Name">
                <p class="form-input-hint">The name is valid.</p>
              </div>

              <div class="form-group has-error">
              <label class="form-label">Select</label>
              <select class="form-select ng-dirty ng-invalid ng-touched">
                ...
              </select>
              <p class="form-input-hint">This option is invalid..</p>
            </div>

            <div class="form-group has-error">
            <label class="form-label">
              <input type="checkbox">
              <i class="form-icon"></i> Send me emails with news and tips
            </label>
          </div>

          <div class="form-group">
            <label class="form-checkbox is-error">
              <input type="checkbox">
              <i class="form-icon"></i> Remember me
            </label>
          </div>

          <div class="form-group">
            <label class="form-radio is-error">
              <input type="radio" checked>
              <i class="form-icon"></i> Male
            </label>
          </div>

          <div class="form-group has-error">
            <label class="form-radio">
              <input type="radio" class="ng-invalid">
              <i class="form-icon"></i> Female
            </label>
          </div>

Disabled styles

Add the disabled attribute to the element or <fieldset> for a disabled form style.


              <div class="form-group">
                <label class="form-label">Name</label>
                <input type="text" class= "form-input" placeholder="Name" disabled>
              </div>

              <div class="form-group">
                <label class="form-label" >Message</label>
                <textarea rows="3" class="form-input" placeholder="textarea example" disabled></textarea>
              </div>

              <div class="form-group">
                <label class="form-label">Select</label>
                <select disabled class="form-select">
                  ...
                </select>
              </div>

              <div class="form-group">
                <label class="form-label">Gender</label>
                <label class="form-radio">
                  <input type="radio" disabled>
                  <i class="form-icon"></i> Male
                </label>

                  <input type="radio" disabled>

                </label>
